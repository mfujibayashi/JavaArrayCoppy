# JavaArrayCoppy
## はじめに

JavaでByte配列からshort配列、Int配列、short配列、Int配列からByte配列に変換するサンプルプログラムです。

初めはエンディアンを考慮してシフト演算をするコードを書きましたが、Javaらしい書き方が有るのではと調べて書いたコードです。

## 参照したページ
